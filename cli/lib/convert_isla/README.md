# Convert Isla

Converts Isla DSL TOML files to ArchSem TOML format.

## Documentation

- **[TOML_SPEC.md](./TOML_SPEC.md)** — ArchSem TOML format specification
- **[TRANSLATION.md](./TRANSLATION.md)** — Isla DSL → ArchSem translation rules

## Usage

```bash
dune exec archsem-convert -- input.toml -o output.toml
```
